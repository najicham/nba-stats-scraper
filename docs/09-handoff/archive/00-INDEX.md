# Session Handoff Archive

## Overview
This directory contains session handoff documents and summaries from previous Claude Code sessions.

## Archive Structure

| Directory | Contents | File Count |
|-----------|----------|------------|
| `2025-11/` | November 2025 sessions | ~30 files |
| `2025-12/` | December 2025 sessions | ~100 files |
| `2025-nov-dec/` | Combined older Nov-Dec 2025 | ~50 files |
| `2026-01-early/` | Early January 2026 (Jan 1-6) | ~80 files |
| `2026-jan-early/` | Early January sessions | ~60 files |
| `2026-jan-mid/` | Mid-January sessions (Jan 7-16) | ~70 files |
| `2026-jan-pre-19/` | Pre-Jan 19 sessions (Jan 16-18 + undated) | 176 files |

## Purpose

These handoff documents served to:
1. Transfer context between Claude Code sessions
2. Document deployment status and pending work
3. Provide quick reference for continuation

## Current Handoff Location

**Active handoffs** (from Jan 19, 2026 onwards) are stored in the main `/docs/09-handoff/` directory.

Naming convention: `YYYY-MM-DD-DESCRIPTION.md`

Examples:
- `2026-01-24-SESSION-HANDOFF.md`
- `2026-01-23-DEPLOYMENT-COMPLETE.md`

## Archival Policy

Handoff documents are archived when:
- Older than 5 days from current date
- Session is complete and action items resolved

**Process:** Move files to dated subdirectory (e.g., `2026-jan-pre-19/`)

**Retention:** Keep archives for 90 days, then review for deletion
