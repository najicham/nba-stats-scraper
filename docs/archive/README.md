# Documentation Archive

**Purpose:** Historical documentation preserved for reference

---

## Structure

```
archive/
├── 2025-11/        # November 2025 archives
├── 2025-10/        # October 2025 archives
├── 2025-08/        # August 2025 archives
├── 2024-10/        # October 2024 archives
└── legacy/         # Very old content (pre-dating date structure)
```

---

## Archive Policy

- Documents move here when superseded or no longer current
- Preserved for historical reference only
- Do not delete - may be needed for context
- New archives go in `YYYY-MM/` format

---

## Finding Archived Content

If looking for a specific document:
1. Check the date it was likely created
2. Look in the corresponding `YYYY-MM/` folder
3. Check `legacy/` for older content
