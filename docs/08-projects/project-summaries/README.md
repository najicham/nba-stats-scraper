# Project Summaries

Periodic snapshots of active project status. Use these to quickly understand the current state of work.

## Usage

- **Latest summary**: Check the most recent dated file
- **Historical context**: Review older summaries to understand project evolution
- **Frequency**: Generate new summary every 1-2 weeks or at major milestones
